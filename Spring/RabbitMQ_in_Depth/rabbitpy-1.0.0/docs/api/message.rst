Message
=======
The :class:`Message <rabbitpy.Messsage>` class is used to create messages that you intend to publish to RabbitMQ and is created when a message is received by RabbitMQ by a consumer or as the result of a :meth:`Queue.get() <rabbitpy.Queue.get>` request.
